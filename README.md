# pwb-semester1-xi-rpl-2

HTML adalah bahasa komputer yang digunakan untuk membuat kerangka atau struktur untuk Web pages (halaman website) di internet. 
HTML adalah singkatan dari Hyper Text Markup Language.
Fungsi HTML adalah sebagai 'kerangka' dari sebuah website.

Macam - macam Tag pada HTML :

```html
<!DOCTYPE Html> Mendefinisikan bahwa dokumen ini adalah html.
<html></html> Tag ini menandai awal dan akhir dari halaman HTML. Semua HTML tag lainnya harus dibungkus dengan tag ini.
<head></head> Pada bagian ini berisi informasi meta tentang dokumen, seperti judul halaman,dan lainnya.
<title></title> Tag ini digunakan untuk menentukan judul halaman yang akan ditampilkan di bilah judul atau tab web.
<body></body> Pada bagian body berisi konten utama dari halaman web, seperti teks, gambar, tautan, dan lainnya.
<h1>, <h2>, <h3>, <h4>, <h5>, <h6> Tag-tag ini digunakan untuk menandai judul dan subjudul dengan tingkatan yang berbeda. Semakin tinggi nomor tag, semakin tinggi tingkat hierkaki judulnya.
<p></p> Tag ini digunakan untuk membuat paragraf teks pada halaman website.
<a></a> Tag ini digunakan untuk membuat tautan. Atribut 'href' digunakan untuk menyimpan link website yang dituju.
<img> Tag ini digunakan untuk menyisipkan gambar di halaman web. Atribut 'src' digunakan untuk menyisipkan dan menampilkan berbagai jenis file eksternal seperti gambar, audio, video, dan lainnya ke dalam halaman web.
<div></div> Digunakan untuk mengelompokkan elemen HTML menjadi blok atau wadah yang terpisah.
<span></span> Digunakkan untuk mengelompokkan elemen HTML menjadi inline atau wadah yang terpisah.
<ul></ul> Merupakan singkatan dari "unordered list" yaitu daftar yang tidak memiliki urutan tertentu, digunakan untuk membuat daftar tak berurutan di halaman web.
<ol></ol> Merupakan singkatan dari "ordered list" yaitu daftar yang memiliki urutan tertentu, digunakan untuk membuat daftar berurutan di halaman web. 
<li></li> Merupakan singkatan dari "list item" digunakan untuk menandai setiap elemen item dalam daftar (baik itu daftar berurutan maupun tak berurutan).
<table></table> Digunakan untuk membuat tabel di halaman web.
<tr></tr> Singkatan dari table row, digunakan untuk mendefinisikan baris pada tabel.
<th></tr> Singkatan dari table header, digunakan untuk mendefinisikan sel header yang biasanya berisi judul atau label untuk setiap kolom.
<td></td> Singkatan dari table data, digunakan untuk mendefinisikan sel data yang berisi informasi sesungguhnya.
<form></form> Digunakan untuk membuat sebuah formulir (form) di halaman web.
<input> Tag ini digunakan untuk menginput teks.
<output> Digunakan untuk menghasilkan hasil inputan.
<textarea></textarea> Digunakan untuk membuat area input text yang lebih besar dalam formulilr.
<button> Tombol ini bisa digunakan untuk memberikan interaktivitas kepada pengguna, seperti mengirimkan formulir, mengaktifkan fungsi JavaScript, atau melakukan tindakan lainnya ketika tombol ditekan.
<select></select> Digunakan untuk membuat elemen drop-down (pilihan tunggal) dalam formulir.
<option></option> Digunakan untuk menandai pilihan dalam elemen drop-down.
<label></label> Digunakan untuk menandai label untuk elemen formulir membantu mengidentifikasi input yang tepat.
<nav></nav> Digunakan untuk menandai blok navigasi situs, seperti menu.
<section></section> Digunakan untuk mengelompokkan konten yang saling terkait secara tematik atau topikal.
<article></article> Digunakan untuk menandai konten yang berdiri sendiri dan dapat dipulikasikan secara independen, seperti postingan blog atau artikel.
<aside></aside> Digunakan untuk menandai konten sampingan atau tambahan yang berhubungan dengan konten utama.
<iframe></iframe> Digunakan untuk menyisipkan halaman web dari situs web lain ke dalam halaman web saat ini.
<meta> Tag ini berisi informasi meta tentang halaman web.
<b> atau <strong> Digunakan untuk membuat huruf menjadi tebal.
<i> Tag ini digunakan untuk membuat huruf bercetak miring.
<video></video> Tag ini digunakan untuk menambahkan video pada halaman web yang kita buat.
<audio></audio> Tag ini digunakan untuk menampilkan suara pada halaman website.
<br> Digunakan untuk membuat baris baru tanpa membuat paragraf baru.
<hr> Digunakan untuk menambahkan garis horizontal untuk memisahkan konten.
<abbr></abbr> Digunakan untnuk menandai singkatan atau akronim dalam teks yang dapat diberikan informasi tambahan saat digunakan dengan atribut.
<cite></cite> Digunakan untuk menandai refrensi ke judul karya seperti buku atau film dalam teks.
<code></code> Digunakan untuk mennadai blok kode komputer atau sintaks.
<pre></pre> Digunakan untuk menandai blok teks yang sudah diformat sebelumnya (preformatted teks), seperti kode komputer.
<blockquote></blockquote> Digunakan untuk menandai kutuipan teks yang panjang, biasanya dengan identasi pada kedua sisi teks. 
<q></q> Digunakan untuk menandai kutipan teks yang pendek, biasanya dalam tanda kutip.
<sub></sub> Digunakan untuk menandai teks sebagai subskrip (teks yang lebih kecil dan posisinya lebih rendah) seperti pada rumus kimia.
<sup></sup> Digunakan untuk menandai teks sebagai superskrip (teks yang lebih kecil dan posisinya lebih tinggi) seperti pada eksponen matematika.
```
